# MrZ's Duplicate Picture Finder

## Explorer mode

Drag all picture files into the window to load them

`Tab` to swtich sorting modes

`Enter` to start filtering

`Del` to generate batch commands which can delete all marked pictures, you need execute the commands manually

`Space` to sort pictures immediately (not neccessary, sorting will be done automatically when entering filter mode)

`Ctrl + Del` to clear the explorer

## Filter mode

All picture pairs which has similar value of crrent sorting mode will be listed

`Left` (`Right`) to mark the picture at the left (right) to be deleted later

`Enter` to back to Explorer mode
